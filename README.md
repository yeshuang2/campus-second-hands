# 校园二手市场

###项目运行环境
1、jdk版本：1.8
3、mysql数据库版本：8.0
4、编译工具：IntelliJ IDEA 2020.1.1
5、数据库可视化工具：Navicat Premium 15
6、maven版本：maven3.6.3

###运行步骤
1、安装jdk1.8,配置环境变量

2、安装mysql8.0,配置环境变量

3、安装Navicat

4、导入ershou.sql数据库文件

5、安装IDEA

6、导入项目concert

7、运行项目

访问url

后台：8080  初始 可用账号 admin  123456

前台：/home/index/index  

初始可用账号：张三 10000/123456   李四：10001/123456
